# Mutual Non-Disclosure Agreement

A mutual non-disclosure agreement (NDA) where both parties agree to keep shared
information confidential for a specified period.

Original template text, not derived from Bonterms or Common Paper standard terms.
Template structure informed by widely used mutual NDA forms.

## Template Variables

| Variable | Type | Default | Description |
|----------|------|---------|-------------|
| `party1` | String | — | Full legal name of the first party |
| `party2` | String | — | Full legal name of the second party |
| `effectiveDate` | DateTime | — | Date the NDA takes effect |
| `purpose` | String | "evaluating a potential business relationship" | Permitted purpose for sharing confidential information |
| `ndaTerm` | String | "1 year" | Duration of the NDA and disclosure period |
| `confidentialityPeriod` | String | "1 year" | How long information remains protected after NDA expires |
| `governingLaw` | String | "Delaware" | State whose laws govern the agreement |
| `jurisdiction` | String | "courts located in New Castle County, Delaware" | Courts with jurisdiction over disputes |
