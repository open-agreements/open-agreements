# OpenAgreements Employment Confidentiality Acknowledgement

Companion acknowledgement form for onboarding packets to capture confidentiality
and approved-use obligations in plain language.

## Source

- **URL**: https://github.com/open-agreements/open-agreements/tree/main/templates/openagreements-employment-confidentiality-acknowledgement
- **Version**: 1.0
- **License**: CC BY 4.0

## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `company_name` | string | yes | Legal name of the company |
| `employee_name` | string | yes | Full legal name of employee |
| `policy_effective_date` | date | yes | Effective date for this acknowledgement |
| `approved_tools_scope` | string | yes | Approved tools/systems scope for company data |
| `data_access_scope` | string | yes | Permitted access scope and business purpose |
| `security_reporting_contact` | string | yes | Contact for reporting incidents or accidental disclosure |
| `post_employment_obligations` | string | no | Obligations after employment ends |
| `acknowledgement_date` | date | yes | Date of employee acknowledgement |
| `signatory_name` | string | yes | Printed name used in signature block |
