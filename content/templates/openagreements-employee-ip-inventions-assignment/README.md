# OpenAgreements Employee IP and Inventions Assignment

Employee proprietary information and inventions assignment agreement designed for
transparent startup hiring and onboarding workflows.

## Source

- **URL**: https://github.com/open-agreements/open-agreements/tree/main/templates/openagreements-employee-ip-inventions-assignment
- **Version**: 1.0
- **License**: CC BY 4.0

## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `company_name` | string | yes | Legal name of employer |
| `employee_name` | string | yes | Full legal name of employee |
| `effective_date` | date | yes | Effective date of this agreement |
| `prior_inventions_disclosure` | string | no | List of prior inventions excluded from assignment, or `None listed` |
| `excluded_inventions_statement` | string | no | Carveout statement for independently developed inventions |
| `confidential_information_definition` | string | yes | Definition summary for confidential information obligations |
| `return_of_materials_timing` | string | yes | Timing for return or deletion of company materials |
| `post_termination_assistance` | string | no | Cooperation obligations after termination |
| `governing_law` | string | yes | Governing law state |
| `venue` | string | yes | Venue for disputes |
